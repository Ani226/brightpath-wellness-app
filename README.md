# brightpath-wellness-app
🌟 BrightPath – Student Wellness & Mood Tracking Web App

BrightPath is a modern, privacy-focused wellness web application designed for students who feel stressed, demotivated, or overwhelmed.
It includes mood tracking, journaling, breathing exercises, analytics, achievements, personalization, and voice-guided support.

✨ All data is stored locally — no login required.

🚀 Features
🎭 Mood Tracking

Quick-tap mood emojis

Soft glow + bounce animation

Saves streak, positive streak, and analytics metrics

Voice feedback (optional)

📊 Analytics Dashboard

Line chart of mood history

Pie chart distribution

Weekly wellness insights

Trend-based wellness scoring

📓 Journal

Write daily reflections

Delete individual entries

Clear entire journal

Auto-timestamp entries

🌬 Breathing Exercises (4-7-8)

Animated breathing circle

Voice-guided breathing

Meditation mode (longer supportive phrases)

Optional background sounds (rain, ocean, flute)

🏆 Achievements System

Streak badges (3/7/14/30 days)

Mood consistency badges

Resilience badge

Confetti celebration & popup

🧑‍🎨 Personalization

Set username

Custom avatar upload

Choose voice for assistant

Optional voice assistant mode

Dark mode toggle

Select daily reminder time

☁ Cloud-Ready

Export all data to JSON

Import backups anytime
